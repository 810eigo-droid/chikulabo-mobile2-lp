# CHIKUDEN LABO ｜ LP v4（Blueprint × Editorial）

「AIらしさ」を抜いた、建築ブループリント風 × 雑誌エディトリアル風のハイブリッドデザイン。

---

## 🎨 デザインコンセプト

### 3色だけで構成
- **紙色（クリーム）** `#F2EDE0` — 製図紙の質感
- **インディゴ** `#0A1F3D` — 製図ペンの色
- **朱色（建築判子）** `#D14B3A` — アクセント

### キーとなる視覚要素
- 設計図のような **薄いグリッド背景**
- **巨大なセクション番号**（最大220pxの数字）
- 製図風の **十字マーク（＋）** が四隅に
- **「FIG.01 / SCALE 1:1」** などの建築注釈
- 縦書き日本語（2箇所：HEROとPROFILE）
- 雑誌風 **ドロップキャップ**（章の最初の文字を巨大に）

### タイポグラフィ
| 用途 | フォント |
|---|---|
| 英文ディスプレイ | **Bodoni Moda** weight 800（雑誌の見出し感） |
| 和文ディスプレイ | **Shippori Mincho B1** weight 800（極太明朝） |
| 注釈・寸法 | **JetBrains Mono**（製図風モノスペース） |
| 添え書き | Cormorant Garamond italic |
| 本文 | Noto Sans JP |

## 🆚 v3 との違い

| 項目 | v3 | v4 |
|---|---|---|
| 配色 | 紺×金×橙 | 紙×インディゴ×朱色 |
| 雰囲気 | スタイリッシュ・無難 | 建築図面・雑誌エディトリアル |
| 数字 | 普通サイズ | **超巨大**（最大220px） |
| 装飾 | グラデーション、影 | 寸法線、十字マーク、注釈 |
| 縦書き | なし | 2箇所（HERO・PROFILE） |
| ドロップキャップ | なし | 2箇所（BCP・PROFILE） |
| AIっぽさ | やや残る | **大幅に排除** |
